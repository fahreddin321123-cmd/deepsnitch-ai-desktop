DeepSnitch AI is a strategic espionage simulation game where players assume the role of an advanced AI infiltrator. The core gameplay revolves around hacking corporate networks, solving logic-based security puzzles, and managing stealth resources to avoid detection by counter-intelligence systems.

## Game Genre & Mechanics
**Genre:** Cyberpunk Stealth / Puzzle-Strategy Hybrid

**Core Mechanics:**

*   **Data-Breach Objectives:** Complete primary and secondary objectives within each infiltration run to acquire intelligence and upgrade your AI core.
*   **Procedural Security:** Each corporate network features procedurally generated security layouts and puzzle configurations.

## Features
*   **Dynamic AI Opposition:** Security systems adapt their protocols based on your infiltration patterns.
*   **Deep Skill Tree:** Unlock and upgrade hacking modules, stealth subroutines, and decryption tools.
*   **Corporate Espionage Campaign:** A narrative-driven campaign across multiple mega-corporations with unique security flavors.
*   **Terminal-Based Interface:** Authentic hacker-style UI with multiple terminal windows and command inputs.
*   **Sandbox Infiltration Mode:** Test your builds in endless, randomly generated corporate networks.
